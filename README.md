Pizza Sales Analysis

Project Goal
Understand pizza sales patterns, peak times, best and worst performers, and category and size contribution to help the restaurant optimise operations, inventory, and promotions.

Tools Used
- SQL Server for KPI calculations and data analysis
- Power BI for interactive dashboard with slicers and filters

Dataset
- Pizza sales records from January to December 2015
- 4 tables: orders, order_details, pizzas, pizza_types

Key Insights
- Classic category contributes the most to total sales and orders
- Large size pizzas are the most popular with almost 46% of sales
- Orders are highest on weekends especially Friday and Saturday evenings
- Busiest months are July, January, and March
- Total Revenue is 817.86K and Total Orders are 21,350
